# 物理层

基本概念
> 电脑要组网，第一件事要干什么？当然是先把电脑连起来，可以用光缆、电缆、双绞线、无线电波等方式。
> 
> 这就叫做”实体层”，逼格高一点的叫法就是物理层。它就是把电脑连接起来的物理手段。它主要规定了网络的一些电气特性，作用是负责传送0和1的电信号。
> 
> 至于0和1的信号是什么，还轮不到物理层还决定。

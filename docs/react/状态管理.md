

# 状态管理

从数据可变性上，主流两种分类

1. 数据可变(mutable)
   - [mobx](/docs/react/mobx)
   - 一些利用proxy API库
2. 数据不可变(immutable)
   - redux
   - redux系列变体


vue中有 数据不可变(immutable)状态管理吗？
 
答案： 几乎没有；

> immutable 适用于函数式编程，vue可以做到函数式，但显然不是最佳实践。
